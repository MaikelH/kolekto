# Kolekto

Collections and datastructures library for Typescript ( and javascript).

## Index

## Installation

```
npm install kolekto 
```

Import the parts you want.

```typescript
import {BloomFilter} from "kolekto"

let filter = new BloomFilter(10000);
```
